# Contributing

Thank you for considering contributing to this module.

- Fork the repository and create a topic branch.
- Write clear, small pull requests with a descriptive title and summary.
- Run `terraform fmt` and `terraform validate` before opening a PR.
- Keep changes centered on a single concern (bug fix, enhancement, docs).
- Add or update examples when behavior changes.
- Maintain backward compatibility or document breaking changes in CHANGELOG.md.

By contributing you agree to follow this repository's Code of Conduct.
